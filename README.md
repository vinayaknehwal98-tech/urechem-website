# Urechem Chemicals Website

Production website for Urechem Chemicals, built with Next.js and deployed on Vercel.

## Current revision

The current production revision includes:

- Updated Urechem Chemicals branding
- “We deliver what we promise” tagline
- Revised full-width polyurethane-solutions hero
- Technical challenge intake on the dedicated challenge page
- Delayed consultation flyer with name, email and mobile fields
- Consultant enquiry flow
- UreShield waterproofing and grouting product-family content

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Verification

```bash
npm run lint
npm run build
```

The production deployment is connected to the repository's `main` branch.
